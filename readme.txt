HTML-Seite, auf der man Memory spielen kann:
 1) Karten, die verdeckt werden können
 2) Das Spiel läuft in Runden 1 Spieler
 3) Pro Runde können 2 Karten aufgedeckt werden
 4.1) Wenn die beiden Karten unterschiedlich sind, dann werden sie wieder zugedeckt, Runde vorbei
 4.2) Wenn die beiden Karten gleich sind, dann bleiben sie aufgedeckt, Runde vorbei 
 5) Wenn alle Karten aufgedeckt sind, ist das Spiel vorbei, je weniger Runden, desto besser

 Step 1: Karten anzeigen (index.html, styles.css)
